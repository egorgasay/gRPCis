# Other

### HISTORY - History of user actions.

Example:
```go
HISTORY
<-
    SET X "Y"
    GET X
```

### SERVERS - List of active servers with stats.

Example:
```go
SERVERS
<- no available servers
```

### ADD SERVER - Add server to the list of active servers.

Example:
```go
ADD SERVER 127.0.0.1:8080
<- OK
```

### EXIT - Exit from the acount. 
```go
EXIT
<- OK
```
