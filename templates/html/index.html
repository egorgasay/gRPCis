Hello, friend!
This is a web view for a single set\get to\from the distributed database called gRPCis.

How it works: Your request will be processed by a web service that will determine the appropriate gRPC method of the memory balancer.
After that, the memory balancer will process your operation by saving\getting the necessary values from\to the storage(-s) and
return you the value (or the error was not found) if it was a get operation or the server number (or the option number that
you specified) on which the value was saved if it was a set operation.

Try it now! Type "help" for more information.

Это веб-приложение для единичной вставки\получения значения распределенной базы данных под названием gRPCis.

Как это работает: Ваш запрос будет обработан веб-службой, которая определит соответствующий gRPC-метод балансировщика памяти.
После этого балансировщик памяти обработает вашу операцию, сохранив\получив необходимые значения из\в хранилище(-ща)
и вернет вам значение (или ошибка не найдено), если это была операция get, или номер сервера (или номер опции, который
вы указали), на котором значение было сохранено, если это была set операция.

Попробуйте сейчас! Введите "справка" для получения дополнительной информации.
